Resources

All the resources I used offered some content which was free of charge. Some offered additional content at various prices. I used, for the most part, the free available content, which was, in my opinion pretty extensive.

  • For quesions:
    • Maybe the best way for getting answers to your questions is to search for them on the search engine of your choice. Most of the time you will find somebody who already had the same question and you will be redirected to perhaps one of the following pages. You obviously can ask questions yourself on these platforms.
    • The probably best known and most used platform concerning programming questions is Stackoverflow.
    • Besides being a great provier for Internet hosting for software development and version control, GitHub is also a great platform for asking and searching for questions.
  • Python:
    • The official python documentation includes a tutorial for beginners.
    • Coding bat provides a set of simple problems for training your python abilities.
    • Mimo is an app for training not only the basics of python, but also the basics of HTML, CSS and JavaScript.
    • freeCodeCamp has a huge catalog of tutorials for various languages including python.
    • There are a ton of channels, playlists, and videos on YouTube that teach python. Some of the ones I used were freeCodeCamp, Tech with Tim and Google Python Classes.
