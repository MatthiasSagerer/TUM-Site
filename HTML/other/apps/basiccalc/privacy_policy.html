BasicCalc - Privacy Policy

BasicCalc was one of the first software projects I did completely on my own, the
first "real" app I created and the first one I released. As it is just a calculator for
basic operations, there isn't any useful data to collect. Therefore, I don't collect any
data with this app.

In case you are interested in how the app works, you can take a look at the code in
the GitHub repository. If you find any bugs in the app, I would appreciate if you
report them in the Repository with an issue. Also, feel free to give suggestion for
improvements or functionality extensions, either in text form or directly as code.

If you're interested in other projects, take a look at the main page.
