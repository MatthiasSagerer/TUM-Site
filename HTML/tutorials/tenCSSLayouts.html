03. Sidebar Says

grid-templatecolumns: minmax(<min>, <max>) ...

Min: 150px / Max: 25%
This element takes the scond grid position (1fr), meaning it takes up the rest of the remaining space.
CSS:
.ex3 .parent {
    display: grid;
    grid-template-columns: minmax(150px, 25%) 1fr;
}
